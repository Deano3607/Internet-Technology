About Internet Technology

Internet Technology is one of the units on the software engineering course at Solent. The general content is aimed at web development; we learn how to design and create websites. The basic content is simple HTML layout but is built upon throughout the year to use advanced CSS to style the pages along with Javascript for interactive on-screen elements.
